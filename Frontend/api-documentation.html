Quick Start Guide

Get up and running with Transalo API in just a few minutes

JavaScript Example

// Install the SDK
npm install @transalo/sdk

// Initialize the client
const transalo = new Transalo({
  apiKey: 'your-api-key',
  region: 'us-east-1'
});

// Translate content
const translation = await transalo.translate({
  text: 'Hello, world!',
  targetLanguage: 'es',
  sourceLanguage: 'en'
});

console.log(translation.text);

Python Example

# Install the SDK

# Initialize the client

    api_key="your-api-key",
    region="us-east-1"
)

# Translate content

    text="Hello, world!",
    target_language="es",
    source_language="en"
